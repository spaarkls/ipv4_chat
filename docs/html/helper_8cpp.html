◆ parse_args()

struct NetworkData parse_args ( int  argc,
char *  argv[] 
)

Парсит аргументы командной строки в структуру NetworkData.

Извлекает IP адрес и порт из массива argv и проверяет корректность IP. Если IP некорректный, выбрасывает исключение BadIp.

Аргументы
argcКоличество аргументов
argvМассив аргументов командной строки (argv[1] - IP, argv[2] - порт)
Возвращает
Структура NetworkData (IP и порт)
Исключения
BadIpЕсли переданный IP адрес, порт или количетсво аргументов невалидно
